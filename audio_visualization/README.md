# 音频简易可视化

## 基本信息

- **项目名称：** 音频简易可视化
- **项目来源：** 原创
- **项目作者：** [LittleYe233](https://github.com/LittleYe233)

## 项目需求

对给定音频可视化，并附有其他设计。

## 主要更改

- 对给定音频可视化为柱状图或波形图；
- 添加音频信息、进度条、背景图等；
- 酌情添加额外特效。
